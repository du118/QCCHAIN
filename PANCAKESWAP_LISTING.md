## ⚠️ 注意事项

### 币价设置

**初始币价建议**: 1 QC = 0.0003-0.001 BNB

**计算公式**：
```
币价 = BNB 数量 / QC 数量
```

**示例**：
- 500,000 QC + 2 BNB
- 币价 = 2 / 500,000 = 0.000004 BNB/QC

---

### 滑点设置

**建议滑点**: 5-10%

**原因**：
- 新代币流动性低
- 滑点太低交易失败
- 滑点太高不划算
