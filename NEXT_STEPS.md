1. 访问：https://pancakeswap.finance
2. 连接钱包
3. 点击 "Liquidity" → "Add Liquidity"
4. 选择：
   - Token A: QC (`0x6b149920B0674fBB86beae9e76724e56AaB0D8b8`)
   - Token B: BNB
5. 注入流动性
