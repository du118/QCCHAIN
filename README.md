## 技术架构

### 核心模块

```
qcchain/
├── consensus/           # 共识机制（PoS + PoW 混合）
├── crypto/             # 抗量子密码算法
│   ├── nist_kyber/     # NIST Kyber KEM
│   ├── nist_dilithium/ # NIST Dilithium 签名
│   └── sphincs_plus/   # SPHINCS+ 无状态签名
├── vm/                 # 虚拟机（兼容 EVM + WASM）
├── cross_chain/        # 抗量子原子跨链协议
├── rwa/                # RWA 真实世界资产模块
├── mining/             # 挖矿与减半机制
├── governance/         # 链上治理
└── wallet/             # 量子钱包核心
```

### 抗量子密码方案

| 组件 | 算法 | 用途 |
|------|------|------|
| 密钥封装 | NIST Kyber-768 | 密钥交换 |
| 数字签名 | NIST Dilithium-3 | 交易签名 |
| 备用签名 | SPHINCS+-128s | 长期安全 |
| 哈希函数 | SHA3-256 | 区块哈希 |
