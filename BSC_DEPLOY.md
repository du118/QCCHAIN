### 2️⃣ 导出私钥

**从 MetaMask 导出**:

1. 打开 MetaMask
2. 点击右上角三个点 → 账户详情
3. 点击 "导出私钥"
4. 输入密码
5. 复制私钥（`0x` 开头，64 位十六进制）

**私钥格式示例**:
```
0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef
```
